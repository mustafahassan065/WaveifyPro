Waveify is a music streaming application built using React Native.
It uses APIs to fetch and display music information and allows users to play music within the app, even in the background.

## Requirements

Node.js must be installed on your system to run this app.

Project Structure

The project contains two main folders:

Assets (Frontend)

Handles the UI and design of the app

Controls how the music and screens are displayed

Backend

Handles the core functionality

Fetches music data from APIs and manages server logic

Installation Steps

You need to install dependencies for both folders.

Step 1: Install dependencies

Open Command Prompt (or Terminal) and run the following command inside both folders:

npm install

Running the Backend

After installing dependencies in the backend folder, run one of the following commands:

node server.js


or

npm run dev

Running the Frontend

After installing dependencies in the assets (frontend) folder, run:

npx expo start


It will take around 30 seconds to compile the files.

After compilation, the Metro Bundler will start.

A QR code will appear in the terminal.

## Running the App

On PC (Localhost):
Press W in the same terminal to run the app in your web browser.

On Mobile Device:

Install the Expo Go app from the Play Store

Scan the QR code using Expo Go

The app will start running on your mobile device
